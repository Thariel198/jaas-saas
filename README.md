# JAAS SaaS

Software de gestión para **Juntas Administradoras de Agua y Saneamiento (JAAS)**, organizaciones comunitarias en Perú encargadas de administrar el servicio de agua potable en sus comunidades.

Este SaaS automatiza los procesos operativos de la JAAS, reduciendo el trabajo manual y mejorando el control del servicio.

## Funcionalidades

- Registro y gestión de usuarios (socios/vecinos)
- Lectura y registro de medidores
- Cálculo automático de consumo y cobranza
- Registro de pagos en efectivo y por Yape
- Generación e impresión de boletas de pago

## Inicio rápido

```bash
# Instalar dependencias
npm install

# Ejecutar en desarrollo
npm run dev
```
