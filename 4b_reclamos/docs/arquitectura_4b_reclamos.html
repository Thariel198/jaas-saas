main.py
Lee
4_pagos/efectivo/outputs/pagos_efectivo.xlsx
outputs/reclamos_YYYY-MM.xlsx (previo, si existe)
outputs/resolucion_reclamos_YYYY-MM.xlsx (si existe, para merge al cerrar)
Escribe
outputs/reclamos_YYYY-MM.xlsx — solo PENDIENTE + EN_REVISION
trazabilidad/trazabilidad_reclamos.xlsx — append RESUELTO/RECHAZADO
Hace
  • Detecta COMENTARIO="reclamo" en pagos_efectivo
  • Preserva RECLAMO / TIPO_RECLAMO / RESOLUCION / ESTADO / FECHA_RESOLUCION
  • Arrastra PENDIENTE/EN_REVISION del mes anterior sin match
  • Cierra RESUELTO/RECHAZADO a trazabilidad con datos de corrección mergeados
resolucion.py
Lee
outputs/reclamos_YYYY-MM.xlsx (con TIPO_RECLAMO ya clasificado)
3_boletas/inputs/DATA_boletas.xlsx (valor vigente — sin snapshot)
Escribe
outputs/resolucion_reclamos_YYYY-MM.xlsx — CAMPO + VALOR_ACTUAL auto-llenados
Hace
  • Cruza (MZ, LT) de reclamos contra DATA_boletas
  • Mapea TIPO_RECLAMO → columna en DATA_boletas → CAMPO
  • Auto-popula CAMPO y VALOR_ACTUAL para el supervisor
  • El supervisor llena VALOR_A_CORREGIR, RESOLUCION, ESTADO

Dependencias externas (solo lectura)

Archivo Ruta Usado por Por qué no se copia a inputs/
pagos_efectivo.xlsx 4_pagos/efectivo/outputs/ main.py Output estable del ciclo previo; no cambia mid-mes
DATA_boletas.xlsx 3_boletas/inputs/ resolucion.py Necesitamos el VALOR_ACTUAL vigente al momento del cruce; snapshot puede quedar desactualizado si 3_boletas regenera mid-mes

Invariantes del módulo

Preservación manual Clave (MESA, MZ, LT, FECHA_COBRO). Nunca sobreescribir lo que el supervisor llenó en re-corridas.
Idempotencia Re-correr sin cambios en inputs no modifica ningún output. Seguro de ejecutar en cualquier momento del mes.
Una historia por reclamo Todo el ciclo vive en una sola fila de trazabilidad_reclamos.xlsx. Sin JOINs para responder "¿qué pasó?".
Corrección null para RECHAZADO CAMPO / VALOR_ANTERIOR / VALOR_APLICADO quedan vacíos si ESTADO_FINAL = RECHAZADO.
Lectura directa de DATA_boletas Siempre el dato vigente. Nunca snapshot. El supervisor ve el valor real al momento de corregir.
No modifica orígenes DATA_boletas.xlsx y pagos_efectivo.xlsx son solo lectura para este módulo.
