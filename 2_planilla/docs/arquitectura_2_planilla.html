02 Planilla — arquitectura

Módulo 2 del pipeline · Generación de planilla mensual de cobro

01 Lecturas → 02 Planilla → 03 Pagos → 04b Validación → 05 Corte → 06 Boletas
Qué hace
Consolidar lecturas + deudas → planilla lista para cobro
Lee las lecturas validadas de 1_lecturas y las une con 4 fuentes de deuda (arrastre anterior, corte, convenios, multas). Calcula el total a pagar por usuario y genera la planilla del mes con columnas de pago vacías listas para que 5_cobranza las llene.
Principio de este módulo
Solo calcula y consolida. No valida lecturas (eso es 1_lecturas). No registra pagos (eso es 5_cobranza). Si un archivo de arrastre no existe → ese campo queda en 0 y el log avisa. El módulo siempre termina.

Clave de join (MZ, LT): MZ → uppercase sin espacios · LT → si es número flotante (1.0) convertir a "1" · si tiene letras conservar uppercase ("11A"). Un usuario del arrastre que no esté en lecturas se ignora (log de advertencia).
Reglas de cálculo
MES_ACTUAL
MES_ACTUAL = max(M3, 5)
Tarifa S/1 por m³ · mínimo de cobro S/5 — si el usuario consumió menos de 5 m³ se cobra 5 igual.
MANTENIMIENTO
MANTENIMIENTO = 3
S/3 fijo para todos los usuarios sin excepción.
TOTAL_A_PAGAR
TOTAL_A_PAGAR = MES_ACTUAL + MANTENIMIENTO + MES_ANTERIOR + CORTE_RECONEXION + CONVENIO + MULTA + ACUERDOS_ASAMBLEA + BLANCO + DEVOLUCION
BLANCO y DEVOLUCION los escribe 5_cobranza como valores negativos — Excel recalcula automáticamente
Pasos internos de main.py
1
Validar y cargar lecturas
Lee inputs/lecturas/lecturas_planilla_YYYY-MM.xlsx. Detecta el mes desde la columna MES_ANO. Normaliza claves (MZ, LT). Falla con mensaje claro si el archivo no existe.
2
Cargar arrastres (tolerante a ausencia)
Intenta leer los 4 archivos de arrastre buscando el YYYY-MM correspondiente. Si un archivo no existe → campo en 0, log de advertencia. No falla.
3
Join por (MZ, LT)
Sobre la base de lecturas, agrega los montos de cada arrastre. Los usuarios sin par en el arrastre quedan en 0 para ese campo.
4
Calcular totales
Para cada usuario:
MES_ACTUAL = max(M3, 5)
MANTENIMIENTO = 3
TOTAL_A_PAGAR = suma de los 6 conceptos
5
Exportar planilla
Escribe outputs/planilla_YYYY-MM.xlsx con las 18 columnas en orden exacto. Columnas de pago vacías. Si el archivo ya existe lo sobreescribe.
